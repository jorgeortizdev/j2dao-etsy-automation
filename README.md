# j2dao-etsy-automation

Automation scripts powering **J2DAO's Etsy + Shopify stores** — from listing creation and SEO optimization to analytics tracking and social content generation.

## What This Does

- **Listing automation** — Generates optimized Etsy product titles, descriptions, and tags using AI
- **SEO pipeline** — Keyword research and on-page optimization for Etsy search ranking
- **Analytics tracker** — Pulls Etsy/Shopify sales data and surfaces top-performing products
- **Social content generator** — Creates Instagram/TikTok captions from product listings
- **Make.com integration** — Triggers automated workflows for new product launches

## Tech Stack

- Python 3.11
- Etsy Open API v3
- Shopify Admin API
- Make.com (automation triggers)
- Ollama (local AI for copy generation)

## About J2DAO

J2DAO Automation is an AI-assisted e-commerce agency. We build and operate print-on-demand and digital product stores with automated content pipelines.
